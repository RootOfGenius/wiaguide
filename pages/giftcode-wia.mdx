# Giftcode WIA

Trang này tổng hợp cách nhập giftcode và danh sách code đang được ghi nhận từ cộng đồng World in Audition.

## Cách Nhập Code

1. Vào **Event Store** trong sảnh chờ.
2. Chọn **Recharge & CDK**.
3. Mở mục **Diamond & CDK**.
4. Copy code, paste vào ô nhập code rồi xác nhận.

![Chọn Recharge & CDK](/images/giftcode-recharge-cdk.png)

![Chọn Diamond & CDK](/images/giftcode-diamond-cdk.png)

![Ô nhập giftcode](/images/giftcode-code-input.png)

## Lưu Ý

- Code có thể hết hạn hoặc hết lượt dùng mà không báo trước.
- Nếu nhập không được, hãy kiểm tra lại ký tự copy/paste và thời hạn sử dụng.
- Một số code trong bảng có ghi chú hạn sử dụng riêng.
